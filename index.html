IOT-INTEGRATED PROJECTS This project is a smart attendance system that uses facial recognition technology integrated with IoT devices. A Raspberry Pi 4 with a camera module captures student images in real-time, recognizes faces using OpenCV, and records attendance automatically. The system updates data instantly in Firebase, and a web dashboard allows admins to view daily attendance, generate reports, and manage student records securely. This solution minimizes manual effort, ensures accuracy, and supports remote monitoring.
